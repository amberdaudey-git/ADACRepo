\## Limitations of the scraper



1. There are limited results per query, because only the first page is fetched.
2. Full article content is not retrieved
3. Scraper only fetches specified languages/countries
4. No error handling for network issues



\## Possible Improvements / Extensions



1. Fetch multiple pages of results
2. Fetch full article content or summaries
3. Expand queries automatically for more coverage
4. Add error handling
